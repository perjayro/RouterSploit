## Description

Module generates payload that creates interactive tcp bind shell for MIPSBE architecture.

## Verification Steps

  1. Start `./rsf.py`
  2. Do: `use payloads/mipsbe/bind_tcp`
  3. Do: `set rport 4321`
  4. Do: `run`
  5. Module generates mipsbe bind shell tcp payload
